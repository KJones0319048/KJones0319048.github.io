Relative Hyperlinks Explained

Recall that when linking to a file located in the same folder or directory, the value of the href attribute is the name of the file.

href="contact.html">Contact

When linking to a file that is inside a folder within the current directory, use both the folder name and the file name in the relative link.

href="rooms/room1.html">Name of the room -- change the filename to reflect the room name

When linking to a file that is up one directory level from the current page, use the “. . / ” notation.

href="../index.html">Home

When linking to a file that is in a folder on the same level as the current folder, the href value will use the “. . / ” notation to indicate moving up one level; then specify the desired folder.

href="../events/weekend.html">Weekend Events
